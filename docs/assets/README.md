# Asset Provenance

`social-preview.png` was generated for the v3.0.0 release on 2026-07-13 with
OpenAI image generation, then resized to GitHub's 1280×640 social-preview ratio
and losslessly/visually compressed for a sub-1 MB repository asset.

Creative direction: a human hand routes a noisy token stream through a precise
prism into a few calm signal lines. It represents the project's core promise:
human judgment stays in charge while deterministic tooling removes waste.

`social-preview.svg` is the previous vector treatment kept for provenance and
comparison. The README and release use the PNG.
